## 📊 Verileri Nasıl Okuyorum?

### 1. Fiyat Değişimi
```
📈 Fiyat Değişimi: +15.42%

Ne Demek:
+15% = SON 5 DAKİKADA %15 YÜKSELDİ (iyi!)
-8% = SON 5 DAKİKADA %8 DÜŞTÜ (kötü, satış baskısı var)

İyi Sinyaller:
+10% ila +20% = Güçlü yükseliş (girilebilir)
+20%+ = Çok güçlü ama geç kalabilirsin

Kötü Sinyaller:
+5% ila +8% = Zayıf sinyal
+50%+ = Çok geç! Muhtemelen dump gelecek
```

### 2. Hacim Artışı
```
📊 Hacim Artışı: x5.5

Ne Demek:
x5.5 = Normalde 5 dakikada 100k$ işlem yapılıyorken şimdi 550k$ yapıldı

İyi Sinyaller:
x5+ = Balina alımı veya haber (çok güçlü!)
x3-5 = Güzel hareket (araştır)

Kötü Sinyaller:
x2-3 = Zayıf (muhtemelen false alarm)
x10+ = Çok aşırı! Dikkat, manipülasyon olabilir
```

### 3. 24h Hacim
```
💵 24h Hacim: $2,450,000

Ne Demek:
Son 24 saatte bu coin için 2.45M$ işlem yapıldı

Güvenilir Hacimler:
$5M+ = Çok popüler coin (BTC, ETH, BNB seviyesi)
$1M-5M = Popüler coin (güvenilir)
$200k-1M = Orta coin (dikkatli ol)
$100k-200k = Küçük coin (riskli)
<$100k = Çok küçük (muhtemelen scam, GİRME!)

NOT: Sistem artık $200k altını görmezden geliyor
```

## 🔍 Adım Adım Değerlendirme

### 1️⃣ Alarm Geldi - İlk 30 Saniye

```
✅ Kontrol Listesi:
□ Sinyal gücü nedir? (🟢/🟡/🔴)
□ Fiyat değişimi +10%'dan fazla mı?
□ Hacim artışı x4'ten fazla mı?
□ 24h hacim $500k'dan fazla mı?

EĞER HEPSİ EVET → Hemen araştırmaya başla
EĞER 2-3 EVET → Araştır ama acele etme
EĞER 1 EVET → Muhtemelen false alarm, geç
```

### 2️⃣ Linklere Tıkla - 1-2 Dakika

#### A) TradingView'da Bak:
```
📊 Grafikte Kontroller:
1. RSI göstergesi:
   - RSI < 70 → GİRİLEBİLİR
   - RSI 70-80 → DİKKATLİ OL (aşırı alım)
   - RSI > 80 → GİRME! (çok aşırı alım, düşecek)

2. Hacim çubukları:
   - Son çubuk diğerlerinden 3-5x büyük → DOĞRULANDI
   - Normal → False alarm olabilir

3. Destek/Direnç:
   - Fiyat direncin altında → GİRİLEBİLİR
   - Fiyat dirençte → BEKLEME (kırmazsa düşer)
```

#### B) Binance'de Bak:
```
💹 Order Book (Emir Defteri):
1. Büyük yeşil duvar (alım emirleri) → İyi sinyal
2. Büyük kırmızı duvar (satış emirleri) → Dikkat, direnç var
3. Son işlemler hep yeşil (alımlar) → Momentum devam ediyor
```

#### C) Twitter/Google Ara:
```
🔍 Aramalar:
- Twitter: "$QKC crypto"
- Google: "QKC coin news today"
- CoinMarketCap: Haberler bölümü

Aramadaki İyi Haberler:
✅ "Binance listing"
✅ "Partnership with [büyük şirket]"
✅ "Mainnet launch"
✅ "Token burn"

Aramadaki Kötü Haberler:
❌ "Scam"
❌ "Rug pull"
❌ "Hack"
❌ "Delisting"
```

### 3️⃣ Karar Ver - 3-5 Dakika

#### ✅ GİR (Alım Yap) - Eğer:
```
1. 🟢 Güçlü sinyal (8-9 puan)
2. Güçlü haber var (listeleme, ortaklık)
3. RSI < 70
4. Fiyat henüz +15% civarı
5. Hacim artışı x5+
```

**GİRİŞ KURALLARI:**
```
💰 Pozisyon: MAX %2 sermaye
📉 Stop-Loss: -%7 (örn: $0.0345'ten gir → $0.0321'de sat)
🎯 Hedef: +%10-15 (örn: $0.0345'ten gir → $0.0380'de sat)
⏰ Zaman: Max 1-4 saat içinde çık
```

#### ⏳ BEKLEME (Gözlemle) - Eğer:
```
1. 🟡 Orta sinyal (5-7 puan)
2. Haber bulamadın
3. RSI 70-80 arası
4. Fiyat zaten +20% artmış
```

**BEKLEME KURALLARI:**
```
👀 Gözlemle: 10-15 dakika izle
📊 Düşerse: Daha iyi fiyattan gir
📈 Yükselirse: Kaçırsan da olur, başka fırsat gelir
```

#### ❌ GİRME (Kaç) - Eğer:
```
1. 🔴 Zayıf sinyal (1-4 puan)
2. Hiç haber yok
3. RSI > 80
4. Fiyat zaten +30%+ artmış
5. 24h hacim çok düşük (<$300k)
6. Twitter'da "scam" yazıları görüyorsun
```

## 🎓 Sık Yapılan Hatalar

### ❌ 1. FOMO Yapmak
```
Yanlış: "Alarm geldi, hemen girmeliyim!"
Doğru: "2-5 dakika araştırayım, sonra karar veririm"
```

### ❌ 2. Her Alarmı Takmak
```
Yanlış: "10 alarm geldi, 10'una da girdim"
Doğru: "10 alarmdan sadece 2-3'ü güçlü sinyal, onlara bakayım"
```

### ❌ 3. Stop-Loss Koymamak
```
Yanlış: "$0.0345'ten girdim, şimdi $0.0250, bekleyeyim yükselir"
Doğru: "Stop-loss $0.0321'de, otomatik satılsın"
```

### ❌ 4. Çok Büyük Pozisyon Açmak
```
Yanlış: "$10k sermayemden $5k QKC aldım!"
Doğru: "$10k sermayemden max $200 riske atabilirim"
```

### ❌ 5. Zaten %30 Artmış Coini Almak
```
Yanlış: "%30 artmış, daha da yükselir!"
Doğru: "Çok geç kaldım, muhtemelen düzeltme gelecek"
```

## 📈 Başarı İçin Kurallar

1. **%2 Kuralı**: Her işlemde MAX portföyünün %2'sini riske at
2. **Stop-Loss**: HER ZAMAN -%7 stop-loss koy
3. **Hedef**: +%10-15'te çık, açgözlülük yapma
4. **Zaman Limiti**: 1-4 saat içinde çık, takılma
5. **Araştır**: 2-5 dakika araştırmadan GİRME
6. **Kayıt Tut**: Her işlemi not et, öğren
